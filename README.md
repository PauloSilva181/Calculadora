# Calculadora

Olá, este é um pequeno projeto que fiz aplicando os conhecimentos que adquiri através do curso "Criação de Projeto em Python com Interface Gráfica" da escola Fundação Bradesco. Para a conclusão do meu objetivo utilizei a linguagem Python para a criação do código juntamente com o Framework Kivy para a criação da Interface de Usuário. Se trata de um projeto muito simples, mas tive vontade de colocá-lo aqui para que possa ver minha evolução ao longo do tempo.
